# PIA Reimagined — HCI Concepts

A modern, accessible redesign of Pakistan International Airlines' web booking experience. This academic HCI project demonstrates an inclusive, responsive UI for flight search, booking, and related services.

## Features

- Accessible, WCAG-aware UI and ARIA attributes
- Mobile-first responsive layout
- Hero with overlapping booking widget
- Rotating promotional banners and featured destinations
- Services cards with dedicated routes
- Awards+ dashboard and utility header/footer

## Tech stack

- TypeScript, React
- Tailwind CSS
- TanStack Router
- lucide-react icons
- Vite / Bun for development

## Run locally

Install dependencies:

```bash
bun install
# or
npm install
```

Start dev server:

```bash
bun dev
# or
npm run dev
```
